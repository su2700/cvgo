## Setup

1. Replace `profile.jpg` with your own photo
2. Edit the content in `index.html` to add your information
3. Open `index.html` in a browser

## Customization

### Update Your Information

Edit these sections in `index.html`:

- **Personal Info** - Name, title, email, phone, location
- **Platforms** - Operating systems you work with
- **Languages** - Programming languages you know
- **Software** - Tools and software you use
- **Media** - Your social media links (LinkedIn, GitHub, etc.)
- **Languages** - Spoken languages and proficiency levels
- **Profile** - Your personality traits
- **Education** - Your educational background
- **Experience** - Your work experience

### Color Scheme

Edit CSS variables in `style.css` to change colors:

```css
:root {
    --color-keyword: #569cd6;  /* Blue for keywords */
    --color-class: #4ec9b0;    /* Teal for classes */
    --color-string: #ce9178;   /* Orange for strings */
    --color-comment: #6a9955;  /* Green for comments */
    /* ... and more */
}
```
